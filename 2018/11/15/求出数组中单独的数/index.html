求出数组中单独的数

Posted on 2018-11-15

题目：

在一个整型数组中，除了一个数以外，其余数均成对存在，找出那个单独的数。
如：input = [ 1, 5, 9, 4, 1, 4, 5] output = 9
最简单的思路就是找到一个数，然后判断有没有另外一个数与之相等。这种思路的代码也很简单，这里我就不再细讲，主要讲另外一种思路。在讲另外一种思路之前，先讲三个重要的常识：

整数编码：

计算机底层存储数据时均采用01编码方式实现，下面分别从非负整数和正整数两个方面介绍：

正整数：

在vc中int型数据占用4个字节32位，也就是说整数由32个0或者1组成，最高位为符号位，0代表正整数，1代表负整数。
所以对于正整数，其存在形式就是其二进制数：如：59表现为00000000 00000000 00000000 00111011，因为59 = 32 + 16 + 8 + 2 + 1

负整数：

对于负整数，-59是不是表现为10000000 00000000 00000000 00111011呢？显然不是，理由之一在于两者相加不等于0。
对于负整数，首先应该求出其对应正整数的二进制编码，然后求出二进制编码的反码，最后求反码的补码，说这么多，举个例子。
对于-59：
编码：00000000 00000000 00000000 00111011
反码就是0变成1，1变成0
反码：11111111 11111111 11111111 11000100
补码就是在反码的基础上加1：
补码：11111111 11111111 11111111 11000101
最后补码就是-59的编码方式。

三种最基本的逻辑电路：

实际上，电脑所有事件的处理都是靠这三种逻辑电路实现的，这里我们不纠结硬件上的问题，只关注逻辑，我们使用0或者1来代表硬件上的电压。

input1 input2 “与”门 “或”门

0 1 0 1
1 0 0 1
1 1 1 1

0 1
1 0

“异或”门

“异或”门是由上面三种基本的逻辑电路组成的，具体构成情况如图：

input1 input2 “异或”门
0 0 0
0 1 1
1 0 1

位操作符

在高级程序语言中，对应逻辑电路运算的操作符为：

逻辑电路 位操作符 描述
“与”门 & 参与运算的两个值，对应位取“与”运算
“或”门 | 参与运算的两个值，对应位取“或”运算
“非”门 ~ 按位取“非”运算
“异或”门 ^ 参与运算的两个值，对应位取“异或”运算

解决方法

  1. 任意两个相同整数之间求“异或”的结果为零。
  2. 任意的整数和零之间求“异或”的结果为其本身。
  3. “异或”运算满足计算的交换律。
